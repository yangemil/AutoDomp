## 一、项目概述

AutoDOMP 是一个功能完整的 Web 应用自动化测试平台,集成了 AI 智能生成、Playwright 测试引擎、定时任务调度等功能。

### 核心特性

1. **AI 智能生成** - 利用 AI 自动生成测试用例和优化选择器
2. **可视化测试场景** - 支持测试用例组合成场景,支持循环执行
3. **定时任务调度** - 支持 Cron 表达式和一次性定时任务
4. **详细的执行报告** - 生成 HTML 格式的测试报告,包含截图和日志
5. **灵活的数据驱动** - 支持 CSV 文件导入测试数据
6. **选择器智能工具** - AI 辅助生成稳定的选择器

## 八、核心业务流程

### 8.1 测试用例执行流程

```
1. 用户发起执行请求
   ↓
2. API 接收请求 (POST /api/execute/:id)
   ↓
3. TestEngineService.executeTestCase()
   ↓
4. 读取测试用例配置
   ↓
5. 创建浏览器实例
   ↓
6. 如果有 CSV 数据,读取并解析
   ↓
7. 遍历测试数据行
   ↓
8. 对于每行数据:
   a. 替换变量 ${变量名}
   b. 执行每个测试步骤
   c. 记录步骤结果
   d. 截图(如果配置)
   e. 记录日志
   ↓
9. 保存执行记录
   ↓
10. 生成测试报告
   ↓
11. 关闭浏览器
   ↓
12. 返回执行结果
```

---

### 8.2 场景执行流程

```
1. 用户发起场景执行请求
   ↓
2. API 接收请求 (POST /api/scenarios/:id/execute)
   ↓
3. 读取场景配置
   ↓
4. 创建场景会话(共享浏览器实例)
   ↓
5. 初始化场景上下文
   ↓
6. 遍历全局循环次数
   ↓
7. 对于每个测试用例:
   a. 读取测试用例配置
   b. 遍历测试用例循环次数
   c. 合并场景上下文和测试数据
   d. 执行测试用例(共享浏览器实例)
   e. 收集测试用例输出的上下文
   f. 更新场景上下文
   g. 记录执行结果
   ↓
8. 关闭场景会话
   ↓
9. 保存场景执行记录
   ↓
10. 生成场景报告
   ↓
11. 返回执行结果
```

---

### 8.3 AI 生成测试用例流程

```
1. 用户输入测试描述
   ↓
2. API 接收请求 (POST /api/ai/generate)
   ↓
3. AIService.generateTestCase()
   ↓
4. 构建提示词(Prompt)
   ↓
5. 调用 AI API
   ↓
6. 解析 AI 返回的 JSON
   ↓
7. 转换为 TestCase 对象
   ↓
8. 保存测试用例
   ↓
9. 返回生成的测试用例
```

---

### 8.4 定时任务执行流程

```
1. TimerService 初始化
   ↓
2. 加载所有定时任务
   ↓
3. 对于一次性任务:
   a. 设置 setTimeout
   b. 到达执行时间后执行场景
   c. 更新任务状态为 completed
   ↓
4. 对于循环任务(Cron):
   a. 使用 node-cchedule 设置定时器
   b. 按照 Cron 表达式定时执行
   c. 记录执行日志
   ↓
5. 任务执行失败时:
   a. 更新任务状态为 failed
   b. 记录错误信息
```

## 十、常见问题与解决方案

### 10.1 端口占用问题

**错误信息**:
```
Error: listen EADDRINUSE: address already in use :::3000
```

**解决方案**:
```bash
# Windows
netstat -ano | findstr :3000
taskkill //PID <进程ID> //F

# Linux/Mac
lsof -ti:3000 | xargs kill -9
```

---

### 10.2 Playwright 浏览器未安装

**错误信息**:
```
Executable doesn't exist at ...
```

**解决方案**:
```bash
npm run install-playwright
```

---

### 10.3 AI 服务连接失败

**错误信息**:
```
AI服务调用失败: ...
```

**解决方案**:
1. 检查系统设置中的 AI 配置是否正确
2. 确认 API 地址和密钥是否有效
3. 使用"测试 AI 连接"功能验证配置

---

### 10.4 CSV 文件导入失败

**错误信息**:
```
无法读取 CSV 文件
```

**解决方案**:
1. 确认 CSV 文件格式正确
2. 检查文件编码是否为 UTF-8
3. 确认数据范围格式正确(如 `1-10` 表示第1到第10行)

---

### 10.5 测试执行超时

**错误信息**:
```
Timeout 30000ms exceeded
```

**解决方案**:
1. 增加项目配置中的超时时间
2. 检查网络连接
3. 检查页面加载速度

---

### 10.6 选择器找不到元素

**错误信息**:
```
未找到元素: ...
```

**解决方案**:
1. 使用"选择器智能工具"优化选择器
2. 检查元素是否动态生成
3. 增加等待时间
4. 使用更稳定的选择器

---

### 10.7 场景执行上下文传递失败

**错误信息**:
```
上下文变量未找到
```

**解决方案**:
1. 确保测试用例正确输出上下文
2. 检查变量名是否正确
3. 查看执行日志确认上下文传递
